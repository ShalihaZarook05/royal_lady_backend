\# Royal Lady Backend



Flask-based backend for the Royal Lady fashion application.



\## Features

\- Product management APIs

\- Recommendation system using ML (TF-IDF + Cosine Similarity)

\- SQLite database

\- Modular Flask Blueprint structure
